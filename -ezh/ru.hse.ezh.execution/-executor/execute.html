execute

fun execute(operations: List<Operation>, globalEnv: Environment): Triple<Int, InputStream, InputStream>
Content copied to clipboard

Performs operations sequentially, passing:

  • output stream as input stream (pipe)

  • status codes

  • local environment (copy of globalEnv that can be modified during execution of operations) from one operation to the next one.

If operations contains a single Assignment, it will be performed on globalEnv rather than on its copy.

If an Exit is encountered, the Environment.exitStatus in globalEnv is set to Environment.ExitStatus.EXITING and the rest of operations are not executed.

Return

Last command exit code, output stream and error stream.

Parameters

operations

The list of operations to execute.

globalEnv

The global session environment.
